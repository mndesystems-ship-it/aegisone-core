# Determinism Proof

Purpose: prove that canonical input produces stable decisions, hashes, and receipts.

Verification:

```powershell
npm run test:local:replay
```

PASS criteria: determinism mismatch rate is `0`, replay drift rate is `0`, and determinism mismatch count is `0`.

Artifacts in `artifacts/` retain compact summaries. Raw generated proof bundles are ignored and regenerated by the command above.
